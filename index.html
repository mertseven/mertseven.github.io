JURY EVALUATION FORM

This is an online jury evaluation form that can be filled for each project. It automatically calculates scores and sends a mail containing all the information in a structured way. It is prepared for New Media Graduation Project I final jury, in 24-25 Fall Semester. You can download the PDF format to fill out physically, or edit the HTML file for your own juries.
