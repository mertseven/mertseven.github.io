JURY EVALUATION FORM

The Jury Evaluation Form is a standardized assessment tool designed for evaluating student projects. With automated score calculation and structured feedback compilation, it streamlines the evaluation process for jury members. The form provides detailed scoring criteria across multiple assessment categories, ensuring comprehensive project evaluation. Upon completion, it automatically generates and sends a structured email report. Currently implemented for the New Media Graduation Project I final jury evaluations in the 2024-25 Fall Semester.

FORTUNE WHEEL

This is an online wheel of fortune tool designed for classroom use. It allows you to randomly select students with a spin animation effect. Names can be added individually, in bulk through text input, or via file upload. The wheel automatically updates its sections for each spin and keeps track of selected students. It features a fullscreen mode for better visibility during class presentations.
