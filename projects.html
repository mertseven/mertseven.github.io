Projects & Explorations

My projects blend together digital media, interactive tools, and educational experiments—each one a curious exploration at the intersection of technology, design, and social understanding. I build these digital playgrounds in collaboration with Anthropic's Claude to investigate how humans (including me) and technology interact, creating systems that reveal patterns in our collective experiences and behaviors.

These works aren't just tools or websites; they're windows into how we communicate, learn, and connect in the digital age. They reflect my fascination with making complex concepts tangible through interactive experiences.

/portfolio/tools-utilities/x-ray-viewer

X-Ray Viewer (Meta)

The X-Ray Viewer is a meta-tool designed to visualize the underlying HTML structure of web elements upon hover. It provides an intuitive way for users to inspect the document object model (DOM), revealing nesting, styling application, and element properties in real-time, aiding in web development and debugging.

Technologies Used: JavaScript (DOM Manipulation) CSS3 Browser Developer Tools Integration (Conceptual)

/portfolio/tools-utilities/curriculum-whiteboard

Curriculum Whiteboard

The Curriculum Whiteboard is an interactive application designed for managing and visualizing university course curriculums. It allows for drag-and-drop course organization across semesters, editing course details (code, name, ECTS, hours, lecturer, syllabus), and tagging courses with customizable categories.

Key features include dynamic ECTS calculation per semester, lecturer workload analysis, and an advanced D3.js-powered network graph to visualize relationships between courses and tags, including co-occurrence links and centrality metrics. The application also supports data import/export for curriculum and workload data.

Technologies Used: HTML5 CSS3 JavaScript D3.js Drag & Drop API Data Visualization
